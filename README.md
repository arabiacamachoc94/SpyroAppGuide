# Guía interactiva: Spyro The Dragon

# 1. Introducción

Se trata de un pequeño tutorial interactivo que te enseña a usar la app de Spyro. Incluyendo elementos multimedia como videos, transiciones o animaciones.

# 2. Características Principales
- **Bienvenida Animada**: Una pantalla de inicio con una línea de carga animada.
- **Guía del Usuario**: Si es la primera vez que entras, una serie de CardViews van explicando cada parte de la App.
- **Listados de Datos**: Secciones con personajes, mundos y coleccionables. Todo se carga desde archivos XML.
- **Efectos y Sonidos**: Animaciones al pulsar elementos y sonidos típicos del juego.
- **Transiciones **: Transición en la primera y última pantalla de la guía , la imagen se desvanece suavemente (Fade).
- **Easter Eggs**: Dos sorpresas incluidas para descubrir.

# 3. Tecnologías y Librerías
Para este proyecto hemos usado :
- **Lenguaje**: Kotlin.
- **Diseño**: XML  para las interfaces.
- **Listas**: RecyclerView con Adapters personalizados.
- **Datos**: XmlPullParser.
- **Animaciones**: ObjectAnimator, ViewPropertyAnimator, ValueAnimator, Interpolator  o ArgbEvaluator.

# 4. Instrucciones de Uso
1. **Descargar**: Descarga el proyecto.
2. **Importar**: Ábrelo con Android Studio.
3. **Ejecutar**: Ejecutar mediante el IDE.
